       float ->
       Evolution.boundary_conditions ->
       Evolution.domain2d -> float -> bool -> t

Function to evolve a wavefunction in time for a given time (in seconds). Returns the internal representation of the wavefunction after the last timestep. Boolean argument tells function whether to print each timestep in a file.
